AWSサービスの全体像と試験範囲

AWSサービスは多数のサービスがあるため、入門者向けのクラウドプラクティショナーの試験であっても対象となるサービスは多岐に渡ります。
対象となるサービスをそれぞれのサービスのジャンルごとにまとめておきます。
詳細についてはまたそれぞれのページにて説明しますので、ざっと確認していきましょう。

1. コンピューティング

アプリケーションを構築する際にサーバーなどのコンピューティングを起動・実行する関連サービスです。

サービス名概要
Amazon EC2AWSを利用して仮想サーバーを立ち上げるサービスです。
関連する機能としてELB／AutoScaling／セキュリティグループといった内容の理解も必須になってきます。
AWS Lambdaサーバレスでプログラミングコード処理を実行するサービスです。
Amazon Lightsail仮想サーバー、ストレージ、データベース、およびネットワーキングを予測可能な低価格で提供します。
Amazon Elastic Container Service
(ECS)		Dockerコンテナをサポートする拡張性とパフォーマンスに優れたコンテナオーケストレーションサービスです。

2. ストレージ

AWSにおいてデータを保存する際に利用する多様なストレージのサービスです。

サービス名概要
Amazon Simple Storage Service
(S3)		中長期に頻繁に利用するデータを大量に保存するためのオブジェクト型ストレージサービスです。
Amazon Elastic Block Store
(EBS)		EC2にネットワークを介してアタッチして利用する専用のブロックストレージです。
インスタンスストアEC2に物理的に接続されている一時データ保存用ストレージです。
Amazon Elastic File System
(EFS)		AWS クラウドサービスおよびオンプレミスリソースで使用するための、シンプルでスケーラブル、かつ伸縮自在な完全マネージド型の NFS ファイルシステムです。
Amazon S3 Glacier安全性と耐久性に優れ、きわめて低コストの Amazon S3 クラウドストレージクラスで、データのアーカイブや長期バックアップに使用します。

3. ネットワーキングとコンテンツ配信

ネットワークやコンテンツ配信・AWS環境への接続設定に利用するサービスです。

サービス名概要
Amazon VPCIPアドレス範囲の選択、サブネットの作成、ルートテーブルやネットワークゲートウェイの設定など、仮想ネットワーキング環境を構築するサービスです。
Amazon CloudFront低レイテンシーの高速転送により世界中の視聴者に安全にコンテンツを配信するコンテンツ配信ネットワーク (CDN) サービスです。
Amazon Route53DNSサーバーの機能を提供し、ドメイン登録とルーティングを実施するサービスです。
AWS Direct ConnectAWS とデータセンター、オフィス、またはコロケーション環境との間にプライベート接続を確立する専用線サービスです。
Amazon API Gatewayリアルタイム双方向通信アプリケーションを実現する RESTfulAPI および WebSocket API を作成・管理します。
AWS Storage Gatewayオンプレミスから実質無制限のクラウドストレージへのアクセスを提供するハイブリッドクラウドストレージサービスです。

4. データベース

AWSでは様々なタイプのデータベースサービスがマネージド型サービスで提供されています。

サービス名概要
Amazon RDSMySQL、PostgreSQL、Oracle、SQL Server、MariaDB 向けのマネージドリレーショナルデータベースサービスです。
Amazon AuroraMySQL および PostgreSQL と互換性のあるクラウド向けの分散・高速化されたリレーショナルデータベースです。
Amazon DynamoDB規模に関係なく数ミリ秒台のパフォーマンスを実現する、keyvalue およびドキュメントデータベースです。
Amazon ElastiCacheRedis または Memcached に互換性のある完全マネージド型のインメモリデータストアです。
Amazon Redshift高速かつシンプルで費用対効果の高い完全マネージド型のデータウェアハウスです。

5. アイデンティティ・セキュリティ・コンプライアンス

AWSでのセキュリティやコンプライアンスに寄与するサービスです。

サービス名概要
AWS Identity & Access Management
（IAM）		AWS のサービスやリソースへのアクセスを安全に管理するアクセス管理サービスです。
Amazon GuardDuty悪意のある操作や不正な動作を継続的にモニタリングする脅威検出サービスです。
Amazon Inspector自動化されたセキュリティ評価サービスで、AWS にデプロイしたアプリケーションのセキュリティとコンプライアンスを向上させることができます。
AWS Key Management Service暗号化キーを簡単に作成して管理し、幅広い AWS のサービスやアプリケーションでの使用を制御できます。
AWS CloudHSM各国のセキュリティ基準に沿った対応を可能にする法令遵守のためのハードウェアベースの暗号化キー用の管理ストレージです。
AWS WAF一般的なウェブの脆弱性からウェブアプリケーションまたは APIを保護するウェブアプリケーションファイアウォールです。
AWS Shieldマネージド型の分散サービス妨害 (DDoS) 保護サービスです。
AWS ArtifactAWS のコンプライアンスレポートにオンデマンドでアクセスできる無料のセルフサービスポータルです。

6. マネジメントとガバナンス

運用保守やサポートに関する支援ツールやサービスです。

サービス名概要
Amazon CloudWatchアプリケーションを監視し、リソース使用率の最適化を行い、運用上の健全性を統括的に把握するモニタリングサービスです。
AWS CloudTrailユーザーアクティビティと API 使用状況の追跡するログ取得サービスです。
AWS ConfigAWS リソースの設定を評価、監査、審査できるサービスです。
AWS Organizations複数のAWS アカウントのまとめて一元管理する機能と一括請求を提供できます。
AWS Personal Health DashboardAWSサービスの状態情報をパーソナライズされた表示を提供する状態管理用ダッシュボードです。
AWS Service CatalogAWS上にデプロイされているサービスの一元的なカタログ上での管理を実施するサービスです。
AWS Well-Architected ToolAWSのアーキテクチャ設計原則の確認とワークロードの見直しと改善を支援します。
AWS Systems ManagerAWS のさまざまなサービスの運用データを確認でき、AWS リソース全体に関わる運用タスクを自動化する運用支援サービスです。
AWS サポートAWSによるサポートサービス
ベーシック／開発者／ビジネス／エンタープライズプランの４つがあります。

7. アプリケーション統合

アプリケーション間の通信制御やフロー作成・データ連携などに利用するサービスです。

サービス名概要
SNS完全マネージド型のpub/sub メッセージングサービスです。
コンポーネント間のメッセージ通知やアラート通知に利用します。
SQS完全マネージド型のメッセージキューイングサービスです。
実行処理の分散並列処理を可能にします。
SESEメールの送受信機能を提供するサービスです。アプリケーション上のEメール通知機能を実装できます。

8. 移行と移転

AWSクラウドへのインフラ移行やデータ移行に利用するサービスです。

サービス名概要
AWS Application Discovery Serviceサーバーの設定データ、使用状況データ、動作データを収集してサーバーの使用率データや依存関係のマッピングなどの移行に必要な情報を提供するサービスです。
AWS Database Migration Serviceデータベースを短期間で安全に AWS に移行することが可能な、データベース移行ツールです。
AWS Server Migration Service数千のオンプレミスワークロードを従来よりも簡単に、かつ短時間で AWS に移行できるエージェントレスサービスです。
AWS Snow ファミリーエッジでデータを収集して処理し、オンプレミスとAWS間でデータを移行する、非常に安全なポータブルデバイスです。

9. AWSコスト管理

運用保守やサポートに関する支援ツールやサービスです。

サービス名概要
AWS Cost ExplorerAWS のコストと使用量や経済性を把握して、AWSコストの分析・管理するための可視化ツールです。
AWSのコストと使用状況レポートAWS のコストと使用状況の詳細を確認するためのレポートです。
AWS Budgets各AWSサービスに応じた予算のしきい値を超えたときにアラートを発信するカスタム予算を設定できます。
価格算定ツールAWS のコスト計算を支援するツールです。
簡易見積りツール/TCO計算ツール/Pricing Calculatorがあります。
AWS Cost Categories自身の組織やプロジェクト構造にわけてコストをカテゴライズして管理することを可能にする機能です。
Trusted Advisorコスト最適化とセキュリティと対障害性とパフォーマンス向上についてアドバイスを提供するサービスです。
