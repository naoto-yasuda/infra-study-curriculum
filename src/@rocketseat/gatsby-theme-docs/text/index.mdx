# インフラエンジニア　学習支援サイト
このサイトはフリースタイルでインフラエンジニアを目指す方の学習支援を目的としたサイトです。  
インフラエンジニアになる上で必要な技術をそれぞれのステップごとに学習を行っていきます。  
左のツリーから各自のステップに応じて学習項目を選択して学習を進めていってください。
## 1. Linuxサーバーの学習
インフラエンジニアとして働いていく上で必須となるLinuxサーバーの操作方法について学習していきます。  
### 1-1. 事前準備
Linuxサーバーの学習をする上で必要になるソフトウェアの準備や、ソフトウェアを使ってのサーバーへのログイン手順について学習します。
### 1-2. コマンド操作
Linuxサーバーを操作する際に利用するコマンドについて、主要なものを学んでいきます。  
Windowsと違ってLinuxサーバーは基本的にコマンドでの操作をしていくことになりますので、それぞれのコマンドの操作方法についてしっかり学んでいきましょう。
### 1-3. 補足
Linuxサーバーを操作する上での補助的な情報やTipsについてまとめています。 
業務をする上で必要になったり、活用できる情報ですので確認していきましょう。
## 2. AWSクラウドプラクティショナー資格学習
現在のインフラエンジニアに求められる技術として最も需要が高いのがAmazon Web Services（AWS）というクラウドサービスに関する技術になります。  
ここではそのAWSの初級資格であるクラウドプラクティショナー資格の取得を目標に、AWSの概要やAWSで利用できるサービスについて学習を行っていきます。
### 2-1. 事前準備
AWSでハンズオン（実習）を行う際に必要なAWSアカウントの発行を行います。
### 2-2. クラウドとAWSの概要
クラウドとは何なのか、AWSとは何なのか等、クラウドやAWSに関する全体的な知識を学んでいきます。
### 2-3. AWSのコアサービス１
AWSで最も利用されるEC2などのサービスについて学習を行います。
### 2-4. AWSのコアサービス２
EC2を利用する上で、合わせて活用されるELBを中心に学習を行います。
### 2-5. AWSのコアサービス３
ストレージやデータベースに関するサービスについて学習を行います。
### 2-6. AWSのコアサービス４
監視機能などAWSの運用をより良くするためのサービスについて学習を行います。
### 2-7. その他の重要サービス
それまでに学習したもの以外で重要なサービスについて学習を行います。
### 2-8. 模擬試験
ここまで学習した内容について模擬テストを行います。
