# AWSアカウントの登録
## 1. AmazonWebServicesのサイトにアクセスする
![](../../aws_cp_account_registration_top_page.png)  
こちらの[リンク](https://aws.amazon.com/jp/free/)をクリックしてAWSのサイトにアクセスしてください。
## 2. サインアップ情報の入力
![](../../aws_cp_account_registration_signup.png)  
メールアドレス、パスワード、アカウント名を入力して「続行」をクリックします。

| 項目 | 入力内容 |
|-------|-------|
| Eメールアドレス | 任意のメールアドレスを入力 |
| パスワード | 大文字・小文字・英字・数字を含んだパスワードを入力する |
| パスワードを確認 | パスワードと同じ内容を入力する |
| AWSアカウント名 | 任意のアカウント名を入力 |
## 3. 連絡先情報の入力
![](../../aws_cp_account_registration_contact_info.png)  
以下のように連絡先情報を入力します。  
連絡先情報は全て英語で入力する必要があるので注意してください。

| 項目 | 入力内容 |
|-------|-------|
| AWSはどの様に使用されますか？ | 個人を選択 |
| フルネーム | 英語でフルネームを入力 |
| 電話番号 | 電話番号を入力 |
| 国または地域コード | 日本を選択 |
| 住所 | 英語で市区町村より後ろの住所を入力 |
| 市区町村 | 英語で市区町村を入力 |
| 州/都道府県または地域 | 英語で都道府県を入力 |
| 郵便番号 | 郵便番号を入力 |
## 4. 請求情報の入力
![](../../aws_cp_account_registration_billing_info.png)  
クレジットカード情報を入力します。  
本カリキュラムの内容は基本的にはAWSのアカウント発行から1年間有効の無料枠を利用しますが、一部無料枠の適用外となり少額課金が発生するものがあります。  
その点を承知の上でカードの登録はお願いします。

| 項目 | 入力内容 |
|-------|-------|
| クレジットカードまたはデビットカード番号 | カード番号を入力する |
| 有効期限日 | カードの有効期限日を入力 |
| カード保有者の氏名 | カード保有者の氏名を入力 |
| 請求先住所 | 連絡先住所と同じで良ければ「連絡先住所を使用する」を選択する |
## 5. 本人確認情報の入力
![](../../aws_cp_account_registration_identification.png)
携帯電話にSMSか音声通話のいずれかで認証用コードを送るように入力し、次のステップで認証を行います。  
なお、このページの手順はSMSを選択した形で記載していきます。

| 項目 | 入力内容 |
|-------|-------|
| 検証コードの受け取り方法 | 任意のメールアドレスを入力 |
| 国または地域コード | 大文字・小文字・英字・数字を含んだパスワードを入力する |
| 携帯電話番号 | パスワードと同じ内容を入力する |
| セキュリティチェック | 表示された文字を入力 |
## 6. 検証コードの入力
![](../../aws_cp_account_registration_identification_input.png)  
携帯電話に届いた検証コードを入力して「続行」をクリックします。
## 7. サポートプランの選択
![](../../aws_cp_account_registration_suport_plan.png)  
無料のベーシックサポートを選択して「サインアップを完了」をクリックします。
## 8. アカウント登録の完了
![](../../aws_cp_account_registration_regist_complete.png)  
「AWSマネジメントコンソールへ進む」をクリックします。
## 9. AWSマネジメントコンソールへのログイン - メールアドレスの入力
![](../../aws_cp_account_registration_login_mail_address.png)  
下記のように選択・入力して「次へ」をクリックします。

| 項目 | 入力内容 |
|-------|-------|
| ユーザー種別 | ルートユーザーを選択 |
| ルートユーザーのEメールアドレス | 登録に利用したEメールアドレスを入力 |
## 10. AWSマネジメントコンソールへのログイン - セキュリティチェックの入力
![](../../aws_cp_account_registration_login_securitycheck.png)  
画面に表示されているセキュリティチェック用の文字列を入力して「送信」をクリックします。
## 11. AWSマネジメントコンソールへのログイン - パスワードの入力
![](../../aws_cp_account_registration_login_password.png)  
登録時に設定したパスワードを入力して「サインイン」をクリックします。
## 12. AWSマネジメントコンソールへのログイン完了
![](../../aws_cp_account_registration_managementconsole.png)  
AWSマネジメントコンソールにログインできたことを確認します。
