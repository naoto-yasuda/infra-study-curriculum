# Macでの操作
MacのTerminalからEC2インスタンスに接続を行う場合は以下の流れで操作をしてください。
## 1. キーペアファイルの移動
以下のコマンドを実行してキーペアファイルを.sshディレクトリに移動させてください。
```shell
mv ~/Downloads/<各自のキーペアファイル名>.pem ~/.ssh/
```
## 2. キーペアファイルの権限変更
以下のコマンドを実行してキーペアファイルの読み取り権限を変更します。
```shell
chmod 400 ~/.ssh/<各自のキーペアファイル名>.pem
```
## 3. インスタンスへの接続
以下のコマンドを実行してEC2インスタンスに接続します。
```shell
ssh -i ~/.ssh/<各自のキーペアファイル名>.pem ec2-user@<EC2インスタンスのパブリックIPv4アドレス>
```
コマンド実行後に接続確認が行われるので「yes」と入力してEnterを押してください。
なお1.～2.の操作は初回接続時のみ行えば大丈夫です。
