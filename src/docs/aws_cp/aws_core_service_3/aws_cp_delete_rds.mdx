# RDSの削除
![](../../aws_cp_delete_rds_delete_db_1.png)
データベースの一覧から対象のインスタンスを選択して、「アクション」-「削除」をクリックします。

![](../../aws_cp_delete_rds_delete_db_2.png)
確認ウィンドウが表示されるので以下のように設定を変更して、入力欄に「delete me」と入力し、「削除」をクリックします。

| 項目 | 設定値 |  |
|-------|-------|-------|
| 最終スナップショットを作成しますか？ | チェックオフ | 削除前にスナップショットを作成するかどうか |
| Retain automated backups | チェックオフ | 自動バックアップを7日間保持するかどうか |
| インスタンスの削除後、システムスナップショットとポイントインタイムの復元を含む自動バックアップが利用不可となることを了承しました。 | チェックオン | 削除に関する確認項目 |

![](../../aws_cp_delete_rds_delete_db_3.png)
データベースの一覧の対象のインスタンスの「ステータス」が「削除中」になっていることを確認します。
