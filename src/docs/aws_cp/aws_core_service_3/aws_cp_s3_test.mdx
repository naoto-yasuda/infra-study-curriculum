### 問題1. <br/>S3の特徴として正しい内容を選択してください。
<details><summary>1. 無制限にデータ容量を利用できる</summary>正解です。<br/>S3は無制限にデータ容量を利用できるストレージであり、容量を設定する必要がありません。</details><br/>
<details><summary>2. アカウント内で一意の名前に設定することが必要である</summary>2.は不正解です。<br/>S3はリージョン内やアカウント内で一意の名前に設定するのではなく、グローバルで全ユーザー間で一意の名称に設定することが必要です。</details><br/>
<details><summary>3. リージョン内で一意の名前に設定することが必要である</summary>3.は不正解です。<br/>S3はリージョン内やアカウント内で一意の名前に設定するのではなく、グローバルで全ユーザー間で一意の名称に設定することが必要です。</details><br/>
<details><summary>4. １つのAZにおける複数サーバーによりデータがレプリケーションされている</summary>4.は不正解です。<br/>AZ内の複数サーバーでデータがレプリケーションされているのではなく、３つ以上の複数のAZに跨ってデータがレプリケーションされています。</details>

### 問題9. <br/>誤って削除してしまったオブジェクトデータの復元方法として正しい内容を選択してください。
<details><summary>1. バージョニングを利用して復元する</summary>正解です。<br/>バージョニングを使用して、Amazon S3バケットに格納されたあらゆるオブジェクトのあらゆるバージョンを、格納、取得、復元することができます。<br/>バージョニングを使用すれば、意図しないユーザーアクションからもアプリケーション障害からも、簡単に復旧できます。</details><br/>
<details><summary>2. ライフサイクル管理により以前のバージョンに戻ることで復元する</summary>2.は不正解です。<br/>ライフサイクル管理はオブジェクトの保存先変更や削除をスケジューリングする機能であり、復元には利用できません。</details><br/>
<details><summary>3. レプリケーションを利用して復元する</summary>3.は不正解です。<br/>S3はクロスリージョンレプリケーションという機能がありますが、バケット間の同期をとるためファイルの削除も同期してしまいます。</details><br/>
<details><summary>4. 自動バックアップを利用して復元する</summary>4.は不正解です。<br/>S3には自動バックアップという機能はありません。</details>
