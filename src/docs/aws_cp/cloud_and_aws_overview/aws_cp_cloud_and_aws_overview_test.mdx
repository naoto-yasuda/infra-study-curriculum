### 問題1. <br/>AWSクラウドコンピューティングを利用する利点はどれですか。
<details><summary>1. 超高速処理</summary>1.は不正解です。<br/>超高速処理はサービスに依存しますが、必ずしもクラウドであれば実現できるというわけではありません。</details><br/>
<details><summary>2. 専有によるセキュリティ向上</summary>2.は不正解です。<br/>専有することは選択によっては可能ですが、基本はインフラを共有することがクラウドの性質になります。</details><br/>
<details><summary>3. 複数の調達サイクル</summary>3.は不正解です。<br/>調達サイクルを短縮化・効率化するのがクラウドの利点です。</details><br/>

### 問題2. <br/>オンプレミス環境とAWS環境を統合した運用を実現するハイブリッド構成のメリットとして間違っている内容を選択してください。

<details><summary>2. AWSクラウドの利点を素早く獲得する</summary>2.は不正解です。<br/>オンプレミス環境から全てクラウドへ移行することは時間がかかるため、部分的にハイブリッドから実行するので迅速にAWSクラウドを活用することができます。</details><br/>
<details><summary>3. 部分的な移行からAWS活用を実現できる</summary>3.は不正解です。<br/>オンプレミス環境から全てクラウドへ移行することは時間がかかるため、部分的にハイブリッドから実行するで迅速にAWSクラウドを活用することができます。</details><br/>
<details><summary>4. 既存のインフラを継続できるため運用管理がやりやすくなる</summary>正解です。<br/>ハイブリッド構成は基本的にはクラウドとオンプレミス環境とを統合した運用管理が必要となり、複雑な対応が必要となります。<br/>統合環境によって効率化するサービスもありますが、いずれにせよ追加の対応が必要です。</details>
