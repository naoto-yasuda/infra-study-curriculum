# AWSの意義
## 1. AWSとは
AmazonWebServices（通称：AWS）は通販で有名なAmazonが提供する総合クラウドサービスで、IaaS・PaaS・SaaSなどあらゆる領域のクラウドサービスを網羅的に提供しています。
## 2. 何故AWSなのか
### 2-1. 業界の圧倒的トップリーダー
![](../../aws_cp_significance_of_aws_cloud_share.png)
画像は2020年のデータですが、AWSは長年クラウド業界のシェアで３割以上をキープしているクラウド業界における絶対王者です。  
![](../../aws_cp_significance_of_aws_gartner_research.png)
またガートナーという世界的なIT分野を中心とするリサーチ＆アドバイザリ企業の行った調査においてもAWSは圧倒的なリーダー的存在と位置づけられています。  
更に日本国内のシェアも元々WindowsOSの利用などで関わりの深かったMicrosoftや、日本企業である富士通・NTTなどを押さえてAWSが大きなシェアを獲得しています。
### 2-2. AWSがトップな理由
AWSを擁するAmazonは研究開発費への投資を長年世界の企業でトップクラスに行い続けています。

| 順位 | 会社名 | 研究開発費 |
|-------|-------|-------|
| 1位 | Amazon | 226億ドル（2.4兆円） |
| 2位 | Google | 162億ドル |
| 3位 | フォルクスワーゲン | 158億ドル |
| 4位 | サムスン | 153億ドル |
| 5位 | インテル | 131億ドル |
| 11位 | トヨタ | 100億ドル |
上記は2018年のデータですが、突出した金額であることが分かると思います。
この圧倒的な技術投資によりIoTやAIなどを含めた最新サービスが続々とAWSでリリースされており、
#### クラウドのスタンダード＝AWSの最新サービス
というような形になっています。  
なのでITインフラを構築することを検討しようとすると、
#### クラウドファースト＝AWSファースト
で検討することとなります。
### 2-3. AWSファースト
上述の通りIT業界ではあらゆる場面でAWSが関わってくるような状態となっています。  
そのためインフラエンジニアはもちろんですが、プログラマーもSEも、果ては営業の方を含めた全てのIT従事者にAWSの知識やスキルが必須となってきています。
