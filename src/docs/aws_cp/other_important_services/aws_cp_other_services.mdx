# その他のサービス
## 1. Codeシリーズ
![](../../aws_cp_other_services_code_series.png)
開発コードのGitベースにリポジトリ上でのコミット・実行・デプロイを自動化する一連のサービスです。  
以下の４つのサービスがあります。
### 1-1. CodePipeline
CodePipelineは、完全マネージド型の継続的なデリバリーが行えるサービスです。
### 1-2. CodeCommit
CodeCommitは、Gitベースのリポジトリをセキュアにホストするマネージド型のソース管理サービスです。
### 1-3. CodeBuild
CodeBuildは、ソースコードをコンパイルし、テストを実行し、デプロイ可能なソフトウェアパッケージを作成できる完全マネージド型のビルドサービスです。
### 1-4. CodeDeploy
CodeDeployは、開発・テスト・本番環境へのデプロイの自動化サービスです。
## 2. Kinesis
Kinesisは、ストリームデータを収集・処理するためのフルマネージド型のサービスで、主に以下の３つのサービスで構成されます。
### 2-1. Amazon Kinesis Data Streams
![](../../aws_cp_other_services_data_streams_image.png)
Amazon Kinesis Data Streamsは、ストリームデータ処理用の分析システムやアプリケーションを構築するサービスです。
### 2-2. Amazon Kinesis Data Firehose
Amazon Kinesis Data Firehoseは、ストリームデータをS3やRedshiftなどへ簡単に配信することが出来るサービスです。
### 2-3. Amazon Kinesis Data Analytics
Amazon Kinesis Data Analyticsは、ストリームデータを標準的なSQLクエリでリアルタイムに可視化・分析出来るサービスです。
## 3. Amazon Athena
![](../../aws_cp_other_services_athena_image.png)
Amazon Athenaは、Amazon S3 内のデータを直接、簡単に分析できるようにするインタラクティブなクエリサービスです。
## 4. Amazon Rekognition
![](../../aws_cp_other_services_rekognition_image.png)
Amazon Rekognitionは、深層学習に基づいた画像認識および画像分析機能を提供するサービスです。
## 5. Amazon Polly
Amazon Pollyは、高度なディープラーニング技術を使用したテキストから音声を合成するサービスです。
## 6. Amazon Lex
Amazon Lexは、音声やテキストを使用して、任意のアプリケーションにチャットボットを構築するサービスです。
## 7. Amazon Transcribe
Amazon Transcribeは、開発者が音声をテキストに変換する機能を提供するサービスです。
## 8. Amazon SageMaker
Amazon SageMakerは、機械学習 (ML) モデルを迅速に構築、トレーニング、デプロイできるようにする完全マネージド型サービスです。
## 9. Amazon QuickSight
![](../../aws_cp_other_services_quicksight_image.png)
Amazon QuickSightは、データを可視化・解析するためのBIツールです。  
RedShiftデータを可視化することが可能です。
## 10. AWS Glue
![](../../aws_cp_other_services_glue_image.png)
AWS Glueは、データを抽出、変換、ロード (ETL) を行う完全マネージド型のサービス
## 11. AWS Lake Formation
![](../../aws_cp_other_services_lake_formation_image.png)
AWS Lake Formationは、本来なら複雑な設定が必要なデータレイクの構成を簡単に素早く実現するサービスです。
## 12. Amazon EMR
![](../../aws_cp_other_services_emr_image.png)
Amazon EMRは、Apache Spark、Apache Hive、Presto などのビッグデータフレームワークを使用して、大量データを処理・分析するサービスです。
