### 問題6. <br/>データを中長期保存するためにコスト最適なストレージを選択しています。<br/>このデータは滅多にアクセスされることはありませんが、必要に応じて１分以内に取得することが必要です。<br/>コスト最適なストレージを選択してください。
<details><summary>1. S3 Standard IA</summary>正解です。<br/>S3 スタンダード-IA は、存続期間が長く、アクセスが頻繁ではないデータ用に設計されています。<br/>これは滅多にアクセスされないファイルを低コストに保存するときに利用します。</details><br/>
<details><summary>2. Glacier（迅速）</summary>2.は不正解です。<br/>Glacierの迅速取り出しでは、データ取得には通常 1〜5 分以内がかかるため、1分以内には対応できません。</details><br/>
<details><summary>3. Glacier Deep Archive</summary>3.は不正解です。<br/>Glacier Deep Archiveはデータ取得に12時間はかかります。</details><br/>
<details><summary>4. S3 Standard</summary>4.は不正解です。<br/>S3 StandardはコストがS3 スタンダード-IA に比較して最適とは言えません。</details>
