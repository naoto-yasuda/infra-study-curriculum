### 問題2. <br/>テラバイト規模のデータ移行をする際に利用するべきサービスを選択してください。
<details><summary>1. Snowball</summary>1.は不正解です。<br/>Snowballでも移行は可能でしたが、現在はSnowball Edge の利用が推奨されています。</details><br/>
<details><summary>2. Snowball Edge</summary>正解です。<br/>Snowball Edge を使用すると、テラバイト規模のデータを 1 週間程度で移動できます。<br/>このサービスを使用することにより、データベース、バックアップ、アーカイブ、健康管理レコード、分析データセット、IoT センサーデータ、メディアコンテンツなどを移動できます。</details><br/>

<details><summary>4. Snowmobile</summary>4.は不正解です。<br/>AWS Snowmobile は、超大容量データを AWS に移動するために使用できるエクサバイト規模のデータ転送サービスです。</details>
