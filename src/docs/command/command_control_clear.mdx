# コマンド操作の学習「clear」
clearコマンドは画面の表示をリセットするコマンドです。  
lsコマンドなど出力の表示量が多いコマンドを実行した際などに利用すると画面がリセットされるため以降のコマンド操作をしやすくすることができます。
## 1. 基本コマンド
画面のリセットは下記のコマンドで行うことができます。
```shell
clear
```
## 2. Tips
### 2-1. ショートカットコマンド
「Ctrl + l（エル）」を入力することでclearコマンドと同様の操作を行うことができます。
## 3. 学習用サーバーに接続して、実際にコマンドを試してみましょう
[リンク](https://bellard.org/jslinux/)を開いて、「x86 Alpine Linux 3.12.0 Console」の「click here」をクリックしましょう。  
出てきたコンソール画面に対してコマンドを入力してみてください。
