# コマンド操作の学習「df」
dfコマンドはサーバーのディスクの空き領域や使用率を表示することができます。  
## 1. 基本コマンド
ディスクの利用情報をただ表示したい場合は下記のコマンドで表示することができます。
```shell
df

※実行結果例
ファイルシス   1K-ブロック    使用   使用可 使用% マウント位置
devtmpfs            479512       0   479512    0% /dev
tmpfs               489592       0   489592    0% /dev/shm
tmpfs               489592     416   489176    1% /run
tmpfs               489592       0   489592    0% /sys/fs/cgroup
/dev/nvme0n1p1    16764908 1944784 14820124   12% /
tmpfs                97920       0    97920    0% /run/user/1000
```
## 2. 便利なdfコマンドのオプション
### 2-1. -kオプション、-mオプション：ディスクの量を指定した単位で表示する
各オプションを利用することで指定した単位でメモリの量を表示することができます。  
キロバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
df -k

※実行結果例
ファイルシス   1K-ブロック    使用   使用可 使用% マウント位置
devtmpfs            479512       0   479512    0% /dev
tmpfs               489592       0   489592    0% /dev/shm
tmpfs               489592     428   489164    1% /run
tmpfs               489592       0   489592    0% /sys/fs/cgroup
/dev/nvme0n1p1    16764908 2319384 14445524   14% /
tmpfs                97920       0    97920    0% /run/user/1000
```
メガバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
df -m

※実行結果例
ファイルシス   1M-ブロック  使用 使用可 使用% マウント位置
devtmpfs               469     0    469    0% /dev
tmpfs                  479     0    479    0% /dev/shm
tmpfs                  479     1    478    1% /run
tmpfs                  479     0    479    0% /sys/fs/cgroup
/dev/nvme0n1p1       16372  2246  14127   14% /
tmpfs                   96     0     96    0% /run/user/1000
```
### 2-2. -hオプション：読みやすい単位で表示する
-hオプションを利用することで自動で読みやすい単位に統一して表示することができます。
```shell
df -h

※実行結果例
ファイルシス   サイズ  使用  残り 使用% マウント位置
devtmpfs         469M     0  469M    0% /dev
tmpfs            479M     0  479M    0% /dev/shm
tmpfs            479M  452K  478M    1% /run
tmpfs            479M     0  479M    0% /sys/fs/cgroup
/dev/nvme0n1p1    16G  2.2G   14G   14% /
tmpfs             96M     0   96M    0% /run/user/1000
```
