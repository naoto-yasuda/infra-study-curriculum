# コマンド操作の学習「echo」
echoコマンドは画面やファイルに文字列や数値、変数の値を表示したり出力したりすることができます。  
色々な場面で使うことの多いコマンドですので、使い方をしっかり理解しましょう。
## 1. 基本コマンド
単純に文字列を表示したい場合は以下のコマンドで表示することが出来ます。
```shell
echo <表示したい内容>

例：echoコマンドで「Hello World!」と表示したい場合
echo Hello World!

※実行結果例
Hello World!
```
変数の中身を表示したい場合は以下のコマンドで表示することが出来ます。
```shell
echo $<変数名>

例：変数aに12345と代入しておいて、変数aの中身を表示する
a=12345
echo $a

※実行結果例
12345
```
文字列と変数を同時に表示したい場合は以下のような形で表示することが出来ます。
```shell
echo number $a OK

※実行結果例
number 12345 OK
```
計算式の結果を表示することも出来ます。
```shell
echo `expr <計算式>`

例：20*2の計算結果を出力する
echo `expr 20 * 2`

※実行結果例
40
```
画面に表示するのではなくファイルに出力する場合は以下のようにコマンドを実行します。
```shell
■新規作成
echo <出力したい内容> > <出力先ファイルパス>

■追記
echo <出力したい内容> >> <出力先ファイルパス>

例：/tmp/testファイルに「test」と出力後、「12345」と追記する
echo test > /tmp/test
echo 12345 >> /tmp/test
```
## 2. 便利なechoコマンドのオプション
### 2-1. -eオプション：エスケープ文字を有効にする
-eオプションを利用することでエスケープ文字を有効にすることができます。  
コマンド入力時に使用できる文字の中で「\」は特殊な扱いになっていてエスケープ文字と呼ばれています。  
このエスケープ文字と特定のアルファベットを組み合わせることで改行など様々な表示をすることが出来ます。  
改行を含んだ文字列を出力する場合は以下のようなコマンドになります。
```shell
echo -e <１行目の文字列>\\n<２行目の文字列>\\n<３行目の文字列>
もしくは
echo -e "<１行目の文字列>\n<２行目の文字列>\n<３行目の文字列>"

「\n」はそのままだと「n」として扱われるため、ダブルクォーテーションで囲まない場合は「\」を二重にする必要があります。

例：「abc」「123」「def」と改行して表示したい場合
echo -e abc\\n123\\ndef
もしくは
echo -e "abc\n123\ndef"

※実行結果例
abc
123
def
```
## 3. クォーテーションによる違い
echoコマンドはクォーテーションの種類によって表示のされ方が変わってきます。
### 3-1. シングルクォーテーション
シングルクォーテーションの場合、クォーテーションの中は全て文字列であると解釈されます。  
例えば「pwd」という文字列を$DIR変数に入れた場合、シングルクォーテーションで囲んで出力すると「$DIR」という文字列が出力されます。
```shell
DIR=pwd
echo '$DIR'

※実行結果例
$DIR
```
### 3-2. ダブルクォーテーション
ダブルクォーテーションの場合、変数が展開されて表示されます。  
例えば同じ様に「pwd」という文字列を$DIR変数に入れた場合、ダブルクォーテーションで囲んで出力すると「pwd」という文字列が出力されます。
```shell
DIR=pwd
echo "$DIR"

※実行結果例
pwd
```
### 3-3. バッククォーテーション
バッククォーテーションの場合、変数内のコマンドが実行されて表示されます。  
例えば同じ様に「pwd」という文字列を$DIR変数に入れた場合、バッククォーテーションで囲んで出力するとpwdの実行結果が出力されます。
```shell
DIR=pwd
echo `$DIR`

※実行結果例
/home/ec2-user
```
