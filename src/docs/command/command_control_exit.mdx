# コマンド操作の学習「exit」
exitコマンドはサーバーからログアウトするためのコマンドです。  
suコマンドでユーザの切り替えを行っている状態でexitコマンドを実行した場合は切り替え前のユーザに戻ります。  
最初にログインしたユーザの状態でexitコマンドを実行するとサーバーから完全にログアウトし、TereTermで接続していた場合はTereTerm自体が終了します。
## 1. 基本コマンド
サーバーからログアウトしたい場合は以下のコマンドを実行することでログアウトすることができます。
```shell
exit
```
## 2. Tips
### 2-1. ショートカットコマンド
「Ctrl + D」を入力することでexitコマンドと同様の操作を行うことができます。
## 3. 学習用サーバーに接続して、実際にコマンドを試してみましょう
[リンク](https://bellard.org/jslinux/)を開いて、「x86 Alpine Linux 3.12.0 Console」の「click here」をクリックしましょう。  
出てきたコンソール画面に対してコマンドを入力してみてください。
