# コマンド操作の学習「free」
freeコマンドはサーバーの物理メモリ、スワップメモリの使用量と空き容量を表示することができます。  
## 1. 基本コマンド
メモリの利用情報をただ表示したい場合は下記のコマンドで表示することができます。
```shell
free

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:         470124      103188      162736         420      204200      354860
Swap:             0           0           0
```
## 2. 便利なfreeコマンドのオプション
### 2-1. -bオプション、-kオプション、-mオプション、-gオプション、--tera：メモリの量を指定した単位で表示する
各オプションを利用することで指定した単位でメモリの量を表示することができます。  
バイト単位でメモリの量を表示する場合は以下の指定です。
```shell
free -b

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:      481406976   107929600   164347904      430080   209129472   361111552
Swap:             0           0           0
```
キロバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
free -k

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:         470124      103280      162616         420      204228      354768
Swap:             0           0           0
```
メガバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
free -m

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:            459         100         159           0         199         346
Swap:             0           0           0
```
ギガバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
free -g
```
テラバイト単位でメモリの量を表示する場合は以下の指定です。
```shell
free --tera
```
### 2-2. -hオプション：読みやすい単位で表示する
-hオプションを利用することで自動で読みやすい単位に統一して表示することができます。
```shell
free -h

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:           459M        100M        158M        420K        199M        346M
Swap:            0B          0B          0B
```
### 2-3. -tオプション：物理メモリとスワップメモリの合計を表す行も表示する
-tオプションを利用することで物理メモリとスワップメモリの合計を表示することができます。
```shell
free -t

※実行結果例
              total        used        free      shared  buff/cache   available
Mem:         470124      103120      162768         420      204236      354928
Swap:             0           0           0
Total:       470124      103120      162768
```
### 2-4. -sオプション：再出力の間隔を指定して再出力されるようにする
-sオプションを利用することでtopコマンドのように指定した間隔で実行結果を表示することができます。
```shell
free -s 5
```
### 2-5. -cオプション：回数を指定して再出力する
-cオプションを利用することで指定した回数だけ実行結果を更新して表示することができます。
```shell
free -c 5
```
