# コマンド操作の学習「less」
lessコマンドはテキストファイルを1画面ずつ表示するためのコマンドです。  
viエディタと似た形でファイルの内容を確認することができます。
## 1. 基本コマンド
```shell
less <ファイル名>
```
testディレクトリのtest.txtファイルをlessコマンドで開く場合は以下のようにコマンドを実行します。
```shell
less test/test.txt
```
## 2. 便利なlessコマンドのオプション
lessコマンドには複数のオプションがあり、それらを活用することでより便利に利用することができます。  
またオプションは複数組み合わせて利用することもできます。
### 2-1. -sオプション：連続した空白行を1行にする
-sオプションを利用することで連続した空白行をまとめた形で出力結果が表示されます。  
空白行の多いファイルを確認する時に活用できます。
```shell
less -s <ファイル名>
```
### 2-2. -Sオプション：画面幅より長い行を折り返さずに表示する
-Sオプションを利用することで画面幅より長い行を折り返さずに表示することができます。  
1行1行が長い横長のファイルを表示する際に活用できます。
```shell
less -S <ファイル名>
```
### 2-3. -Nオプション：行番号を表示する
行番号を表示したい場合は-Nオプションを利用します。
```shell
less -N <ファイル名>
```
## 3. lessで表示中の便利なキー操作
lessコマンドでファイル内容を表示している時はviエディタと同じ様にキー操作で
またオプションは複数組み合わせて利用することもできます。
### 3-1. qキー：less表示を修了する
lessの表示を終了したい場合はqキーを利用します。
### 3-2. vキー：viエディタを起動する
lessで表示しているファイルをそのまま編集したい場合はvキーを利用します。
### 3-3. !キー：シェルでコマンドを実行する
lessで表示しているファイルをそのままシェルで実行したい場合は!キーを利用します。
### 3-4. /文字列キー：ファイル内を検索する
viエディタと同様にファイル内の特定の文字列の場所へ移動したい時は「/文字列」を入力します。  
WindowsにおけるCtrl+Fと同様に利用することができます。  
「/文字列」を実行した後は「n」キーで次の該当文字列の場所へ移動できます。  
また「N」キーで前の該当文字列の場所へ移動できます。
