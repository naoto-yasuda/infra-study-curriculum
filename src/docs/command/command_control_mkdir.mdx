# コマンド操作の学習「mkdir」
mkdirコマンドはディレクトリを新規作成するコマンドです。  
Windowsで言うところの「右クリック→新規作成→フォルダー」の操作と同じイメージです。
## 1. 基本コマンド
ディレクトリを新規作成するだけなら下記のコマンドで行うことができます。
```shell
mkdir <ディレクトリ名>

例：testディレクトリを作成したい場合
mkdir test
```
## 2. 便利なmkdirコマンドのオプション
### 2-1. -mオプション：権限を指定してディレクトリを作成する
-mオプションを利用することでディレクトリの権限を指定した状態で作成することができます。  
システムの要件などで所定の権限のディレクトリを作成したい場合に活用できます。
```shell
mkdir -m <権限指定> <ディレクトリ名>

例：testディレクトリを全ユーザに全権限を付与して作成したい場合
・権限指定を数字で指定する場合
mkdir -m 777 test
・権限指定をアルファベットで指定する場合
mkdir -m a=rwx test
```
※権限指定の詳細は[権限の指定方法について](../linux_control/permission)を参照してください
### 2-2. -pオプション：エラーを表示せずに指定したディレクトリが存在しなければ作成する
-pオプションを利用することで存在しないディレクトリに対してディレクトリを作成しようとした際にエラー無く、存在しない親ディレクトリも含めて作成をすることができます。  
このオプションを指定しないと親ディレクトリが存在しない場合はエラーメッセージが表示されます。  
複数階層のディレクトリ構成を一気に作りたい場合に便利に利用できます。
```shell
mkdir -p <親ディレクトリ名>/<ディレクトリ名>

例：test1ディレクトリを新規作成、test1ディレクトリ配下にtest2ディレクトリを新規作成、
　　test2ディレクトリ配下にtest3を新規作成したい場合
mkdir -p test1/test2/test3
```
### 2-3. -vオプション：ディレクトリを作成した結果のメッセージを表示する
-vオプションを利用することでコマンド実行に作成したディレクトリの結果を表示することができます。  
１つディレクトリを作成するだけなら利用しなくてもそこまで困りませんが、-pオプションを利用して複数階層のディレクトリを一気に作成した場合に一度に作成確認ができるので便利です。
```shell
mkdir -v <ディレクトリ名>

例：test1ディレクトリを新規作成、test1ディレクトリ配下にtest2ディレクトリを新規作成、
　　test2ディレクトリ配下にtest3を新規作成して結果メッセージを表示する場合
mkdir -p -v test1/test2/test3

※出力結果例
mkdir: ディレクトリ 'test1/' を作成しました
mkdir: ディレクトリ 'test1/test2/' を作成しました
mkdir: ディレクトリ 'test1/test2/test3/' を作成しました
```
## 3. Tips
### 3-1. オプションは同時指定可能
2-3.のコマンド例で利用していますが、オプションは複数同時に指定して実行することができます。
