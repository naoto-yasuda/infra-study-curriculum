# コマンド操作の学習「rm」
rmコマンドはファイルやディレクトリを削除するコマンドです。  
Windowsで違ってゴミ箱の概念が存在しないので削除の操作は慎重に行いましょう。
## 0. 事前準備
### ※コマンド試し用ファイルの準備
本ページのコマンドを試すためのファイルは以下のコマンドを一通り実行することで作成することができます。
```shell
mkdir test
touch test/test01
touch test/test02
touch test/test03
```
## 1. 基本コマンド
ファイルを削除する場合は下記のコマンドで行うことができます。
```shell
rm <ファイル名>

例：testディレクトリのtest01ファイルを削除したい場合
rm test/test01
```
ワイルドカードを使ってディレクトリ配下のファイルを一括で削除したり、条件に合うファイルを一括で削除することもできます。
```shell
例：testディレクトリ配下のファイルを削除したい場合
rm test/*
※ドットから始まるファイルは削除されないので注意

例：testディレクトリのtestから始まるファイルを削除したい場合
rm test/test*
```
## 2. 便利なrmコマンドのオプション
### 2-1. -iオプション：ファイルの削除前に確認する
-iオプションを利用することでファイルの削除前に確認を行うようにすることができます。  
最初に書いたようにLinuxでは削除すると元に戻す事はできないため、慎重に削除を行いたい際に活用できます。
```shell
rm -i <ファイル名>

例：testディレクトリのtest01ファイルを確認ありで削除したい場合
rm -i test/test01

※実行結果例
以下のメッセージが表示される
rm: 通常の空ファイル 'test/test01' を削除しますか?

メッセージ表示後、「y」または「Y」を入力してEnterキーを押すことでファイルが削除される
それ以外の文字を入力するか、何も入力せずにEnterキーを押すことでファイルを削除せずに
処理を修了する
```
### 2-2. -rオプション：ディレクトリも削除の対象とする
-rオプションを利用することでrmコマンドでディレクトリを削除できるようになります。  
-rオプションを付けずにディレクトリを削除しようとするとエラーメッセージが表示されます。
```shell
rm -r <ディレクトリ名>

例：testディレクトリを削除したい場合
rm -r test
```
### 2-3. -vオプション：削除の詳細を表示する
-vオプションを利用することでどのファイルやディレクトリが削除されたかを表示することができます。  
削除した内容をログや証跡として残したい時に活用できます。
```shell
rm -v <ファイル名もしくはディレクトリ名>

例：testディレクトリを削除して、削除の詳細を表示したい場合
rm -r -v test

※実行結果例
'test/test01' を削除しました
'test/test02' を削除しました
'test/test03' を削除しました
ディレクトリを削除しました: 'test/'
```
## 3. Tips
### 3-1. オプションは同時指定可能
2-3.のコマンド例で利用していますが、オプションは複数同時に指定して実行することができます。
