# コマンド操作の学習「sudo」
sudoコマンドはユーザを切り替えずに他のユーザの権限でコマンドを実行するためのコマンドです。  
suコマンドでユーザを切り替える程作業をしない際などわざわざユーザを切り替えずにコマンド実行できるため便利です。
## 0. 事前準備
### ※テスト用ユーザの追加
以下のコマンドを実行してテスト用のユーザを作成します。
```shell
useradd testuser
```
## 1. 基本コマンド
rootユーザでコマンドを実行したい場合は以下のコマンドを実行することでコマンド実行することができます。
```shell
sudo <コマンド>

※コマンドを実行するとパスワードが求められるので、パスワードを入力することで
　指定のユーザでコマンドが実行されます

例：rootでwhoamiコマンド（現在のユーザ名を表示するコマンド）を実行する場合
sudo whoami

※実行結果例
root
```
## 2. 便利なsudoコマンドのオプション
### 2-1. -uオプション：指定したユーザでコマンドを実行する
-uオプションを利用することで指定したユーザでコマンドを実行することができます。  
```shell
su -u <ユーザ名> <コマンド>

例：testuserでwhoamiコマンド（現在のユーザ名を表示するコマンド）を実行する場合
sudo -u testuser whoami

※実行結果例
testuser
```
