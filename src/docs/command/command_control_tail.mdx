# コマンド操作の学習「tail」
tailコマンドはファイルの最終行から標準だと１０行を表示するためのコマンドです。  
catやlessとはまた違う用途でファイルの内容を確認することができます。
## 1. 基本コマンド
```shell
tail <ファイル名>
```
testディレクトリのtest.txtファイルをtailコマンドで開く場合は以下のようにコマンドを実行します。
```shell
tail test/test.txt
```
## 2. 便利なtailコマンドのオプション
tailコマンドには複数のオプションがあり、それらを活用することでより便利に利用することができます。  
またオプションは複数組み合わせて利用することもできます。
### 2-1. -cオプション：出力する文字数を指定する
-cオプションを利用することで出力する文字数を指定することが出来ます。  
```shell
tail -c <出力文字数> <ファイル名>

例：/etc/servicesの末尾から１０文字を表示する
tail -c 10 /etc/services

※出力結果例
ri Broker
```
### 2-2. -nオプション：出力する行数を指定する
-nオプションを利用することで出力する行数を指定することが出来ます。  
大量行のファイルの最終行から見たい行数を見ることが出来ます。
```shell
tail -n <出力行数> <ファイル名>

例：/etc/servicesの末尾から１０行を表示する
tail -n 10 /etc/services

※出力結果例
3gpp-cbsp       48049/tcp               # 3GPP Cell Broadcast Service Protocol
isnetserv       48128/tcp               # Image Systems Network Services
isnetserv       48128/udp               # Image Systems Network Services
blp5            48129/tcp               # Bloomberg locator
blp5            48129/udp               # Bloomberg locator
com-bardac-dw   48556/tcp               # com-bardac-dw
com-bardac-dw   48556/udp               # com-bardac-dw
iqobject        48619/tcp               # iqobject
iqobject        48619/udp               # iqobject
matahari        49000/tcp               # Matahari Broker
```
### 2-3. -fオプション：ファイルの追記を監視する
ファイルの追記を監視したい場合は-fオプションを利用します。  
ログファイルに出力されるログをリアルタイムで確認するのに便利です。
```shell
tail -f <ファイル名>
```
