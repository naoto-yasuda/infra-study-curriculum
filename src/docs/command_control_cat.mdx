# コマンド操作の学習「cat」
catコマンドはLinuxサーバーでファイルの中身を確認するために利用するコマンドです。 
設定ファイルの内容を確認したり、ログファイルのログを確認したりなど、コマンドの中でも特によく使うコマンドになります。  
## 1. 基本コマンド
catコマンドの後ろに中身を見たいファイルを指定して実行することでファイルの中身を確認できます。
```shell
cat <ファイル名>

cat File_A

※出力結果例
aaa
bbb
ccc



ddd
eee
fff
```
## 2. 応用コマンド
catコマンドは使い方によってファイルの中身の確認以外にも便利な使い方があります。
### 2-1. ファイル内容のコピー
コマンドの「>」を使うとコマンドの出力内容をファイルに出力することができます。  
それを利用してcatコマンドの出力内容を別ファイルに出力することでファイルのコピーをすることができます。
```shell
cat File_A > File_B
```
出力先に指定したファイル名が存在していても、していなくても新しいファイルとして作成されるので注意しましょう。
### 2-2. ファイル内容の他ファイルへの追記
コマンドの「>>」を使うとコマンドの出力内容をファイルに追記することができます。  
それを利用してcatコマンドの出力内容を別ファイルに追記することができます。
```shell
cat File_A >> File_B
```
出力先に指定したファイルが存在していない場合は新しいファイルとして作成されます。  
存在している場合はファイルへと出力内容が追記されます。
### 2-3. ファイルの結合
２つ以上のファイルを結合して出力したい場合にもcatコマンドが利用できます。
```shell
２つの場合
cat File_A File_B

３つの場合
cat File_A File_B File_C
```
上記のコマンドだけだとコマンドの実行結果として表示されるだけなので、ファイルに出力したい場合は「>」「>>」などのコマンドを利用します。
```shell
新規作成
cat File_A File_B > File_C

追記
cat File_A File_B >> File_C
```
## 3. catコマンドのオプション
### 3-1. -nオプション：行番号を付けて出力する
-nオプションを利用することで出力結果に行番号が表示されます。  
設定ファイルのエラー調査時など利用機会が多く便利なオプションなので活用していきましょう。
```shell
cat -n File_A

※出力結果例
1 aaa
2 bbb
3 ccc
4 
5 
6 
7 ddd
8 eee
9 fff
```
### 3-2. -bオプション：空白行を入れずに行番号を付けて出力する
-bオプションを利用することで出力結果に空白行を除いて行番号が表示されます。  
ログの出力量をカウントしたい時などに利用できます。
```shell
cat -b File_A

※出力結果例
1 aaa
2 bbb
3 ccc



4 ddd
5 eee
6 fff
```
### 3-3. -sオプション：連続した空白行を１行の空白行にまとめて出力する
-sオプションを利用することで連続した空白行をまとめた形で出力結果が表示されます。  
空白行の多いファイルを確認する時に活用できます。
```shell
cat -s File_A

※出力結果例
aaa
bbb
ccc

ddd
eee
fff
```
### 3-4. -Eオプション：各行の最後に"$"を出力する
-Eオプションを利用することで各行の最後に"$"が出力されます。  
各行の最後をわかりやすい状態でファイルを確認したい時に使えます。
```shell
cat -E File_A

※出力結果例
aaa$
bbb$
ccc$
$
$
$
ddd$
eee$
fff$
```

### 3-5. -Tオプション：タブを"^|"に置き換えて出力する
-Tオプションを利用することでタブを"^|"に置き換えて出力されます。  
タブをわかりやすい状態でファイルを確認したい時に利用すると良いでしょう。
```shell
cat -T File_A

※出力結果例
aaa
^|bbb
^|^|ccc
^|^|^|ddd
```
## 4. Tips
### 4-1. ファイルの表示順を逆にしたい
catコマンドを逆さにしたtacというコマンドで表示順を逆にして表示できます。  
ただオプションは異なるため単純に出力内容を逆にしたい時に利用しましょう。
```shell
tac <ファイル名>
```
