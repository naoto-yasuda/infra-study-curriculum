# コマンド操作の学習「cd」
cdコマンドはLinuxサーバー上で自分が今いるフォルダを移動するためのコマンドです。  
Windowsのエクスプローラでフォルダをダブルクリックしたり、上へを押したりして移動するのと同じようなことができます。
## 1. 基本コマンド
```shell
cd <移動先のパス>
```
## 2. 特殊なパス指定
| コマンド | 意味 |
| ------- | -------|
| ~ | ホームディレクトリ（ユーザごとに割り当てられるディレクトリ） |
| ./ | 現在のディレクトリ |
| .. | １つ上のディレクトリ |
## 3. コマンドの入力例
### 3-1. フォルダ構成の例
```
/
└─ home
     └─ ec2-user → ホームディレクトリ（現在のディレクトリでもある）
          ├─ user
          │   ├─ suzuki
          │   ├─ yamada
          │   └─ tanaka
          └─ file
               ├─ aaa
               ├─ bbb
               └─ ccc
```
### 3-2. userディレクトリへ移動する
#### 相対パスでの移動
```shell
cd user
```
もしくは
```shell
cd ./user
```
#### 絶対パスでの移動
```shell
cd /home/ec2-user/user
```
### 3-3. suzukiディレクトリへ移動する
#### 相対パスでの移動
```shell
cd user/suzuki
```
もしくは
```shell
cd ./user/suzuki
```
#### 絶対パスでの移動
```shell
cd /home/ec2-user/user/suzuki
```
### 3-4. 現在のユーザのホームディレクトリへ移動する
```shell
cd
```
もしくは
```shell
cd ~
```
### 3-5. 上の階層へ移動する
```shell
cd ..
```
### 3-6. ２つ上の階層へ移動する
```shell
cd ../..
```
### 3-7. tanakaディレクトリにいる状態でcccディレクトリに移動する
#### 相対パスでの移動
```shell
cd ../../file/ccc
```
#### 絶対パスでの移動
```shell
cd /home/ec2-user/file/ccc
```
## 4. Tips
### 4-1. 入力補完
パスを入力する際、Tabキーを押すことでディレクトリ名やファイル名を補完することができます。
