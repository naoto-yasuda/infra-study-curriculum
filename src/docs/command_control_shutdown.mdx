# コマンド操作の学習「shutdown」
shutdownコマンドは文字通りサーバーをシャットダウンするためのコマンドです。  
シャットダウンだけでなくオプションを利用することで再起動もすることができます。
## 1. 基本コマンド
shutdownコマンドの基本的な形は以下になりますが、基本的にはオプションを付与して実行します。
```shell
shutdown <時間>

※時間の指定方法は以下の形があります
　・今すぐにシャットダウンする場合：now
　・シャットダウンまでの時間で指定する場合：+<分数>で分数を指定する
　・指定時刻でシャットダウンする場合：hh:mm形式で時分を指定する

例：今すぐシャットダウンする場合
shutdown now

例：5分後にシャットダウンする場合
shutdown +5

例：19時にシャットダウンする場合
shutdown 19:00
```
## 2. 便利なshutdownコマンドのオプション
### 2-1. -hオプション：サーバーの電源を切る
-hオプションを利用することでサーバーの電源を切る形でシャットダウンすることができます。  
基本的にshutdownコマンドを使ってサーバーをシャットダウンする際はこの-hオプションを付与してコマンドを実行します。
```shell
shutdown -h <時間>

例：今すぐ電源を切る形でシャットダウンする場合
shutdown -h now
```
### 2-2. -rオプション：サーバーを再起動する
-rオプションを利用することでサーバーの再起動をすることができます。  
```shell
shutdown -r <時間>

例：今すぐ再起動する場合
shutdown -r now
```
### 2-3. -cオプション：動作中のshutdownコマンドをキャンセルする
-cオプションを利用することでshutdownコマンドを時間指定して実行後シャットダウンを待っている間にシャットダウンのキャンセルをすることができます。  
時間指定をnowで実行した場合は即時にシャットダウンが開始するのでキャンセルはできないが、それ以外の場合はシャットダウンが行われるまでの間はキャンセルすることが可能です。
```shell
shutdown -c
```
またshutdownコマンドを実行したTeraTermなどの画面からであればCtrl+Cを実行することでもキャンセルすることができます。
