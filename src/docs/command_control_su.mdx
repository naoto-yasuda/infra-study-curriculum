# コマンド操作の学習「su」
suコマンドはLinuxサーバー上で利用するユーザを切り替えるためのコマンドです。  
Windowsのようにログインし直す形ではなく、コマンドを実行するユーザが切り替わる形になります。
## 0. 事前準備
### ※テスト用ユーザの追加
以下のコマンドを実行してテスト用のユーザを作成します。
```shell
useradd testuser
```
## 1. 基本コマンド
指定のユーザに切り替えたい場合は以下のコマンドを実行することで切り替えることができます。
```shell
su <ユーザ名>

※コマンドを実行するとパスワードが求められるので、パスワードを入力することで
　指定のユーザに切り替わります
```
ユーザ名の指定をしない場合、rootユーザに切り替えることができます。
```shell
su
```
## 2. 便利なsuコマンドのオプション
### 2-1. -オプション：現在のシェルの環境変数をすべて解除し、指定したユーザとしてシェルを起動する
-オプションを利用することで現在のユーザで設定された環境変数をすべて解除し、指定したユーザとしてシェルを起動することができます。  
オプション無しだとあくまでユーザを切り替えるイメージですが、-オプションを付けることで指定したユーザでログインし直すようなイメージになります。
```shell
su - <ユーザ名>

※ユーザ名の指定をしない場合、オプション無しと同様rootユーザに切り替えることができます

例：testuserに切り替えたい場合
su - testuser
```
## 3. Tips
### 3-1. 切り替え前のユーザに戻したい場合
切り替え前のユーザに戻したい場合は[exit](/command_control_exit)コマンドで戻すことができます。
