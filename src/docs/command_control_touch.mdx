# コマンド操作の学習「touch」
touchコマンドは本来はファイルのタイムスタンプを変更するコマンドですが、存在しないファイル名を指定することで簡単にファイルを新規作成することができます。  
中身は空でも良いので特定の名前のファイルを準備する必要がある時などに利用されます。  
## 1. 基本コマンド
ファイルのタイムスタンプを変更したい場合は、touchコマンドの変更したいファイルを指定して実行することでファイルのタイムスタンプを変更できます。  
何もオプション無しで実行すると現在時刻に変更することができます。
```shell
touch <ファイル名>

例：testディレクトリのtest.txtファイルのタイムスタンプを現在時刻に変更したい場合
touch test/test.txt
```
test.txtファイルが存在しない場合は新規作成されます。
## 2. 便利なtouchコマンドのオプション
### 2-1. -dオプション：日時を指定してファイルのタイムスタンプを変更する
-dオプションを利用することで指定した日時にファイルのタイムスタンプを変更することができます。  
設定ファイルのエラー調査時など利用機会が多く便利なオプションなので活用していきましょう。
```shell
touch -d <日付> <ファイル名>

例：testディレクトリのtest.txtファイルのタイムスタンプを「2021-09-15 20:00」に変更したい場合
touch -d "2021-09-15 20:00" test/test.txt
```
### 2-2. -rオプション：他のファイルのタイムスタンプに合わせる
-rオプションを利用することで既存のファイルのタイムスタンプと同じタイムスタンプに変更することができます。  
そこまで利用する機会は無いかもしれませんが、ひとかたまりのファイル群の最新の更新日時を合わせて分かりやすくしたい時などに活用できます。
```shell
touch -r <既存のファイル名> <対象ファイル名>

例：testディレクトリのtest_a.txtファイルのタイムスタンプをtest_b.txtと同じタイムスタンプに変更したい場合
touch -r test/test_b.txt test/test_a.txt
```
