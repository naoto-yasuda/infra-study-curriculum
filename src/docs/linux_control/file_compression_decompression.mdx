# ファイルの圧縮・解凍
LinuxサーバーでもWindowsと同じ様にファイルの圧縮・解凍を行うことが出来ます。  
Windowsでは.zip形式の圧縮ファイルが基本ですが、Linuxの場合では.tarもしくは.tar.gzの形式のファイルを扱うことが多いです。  
ここではLinuxで扱う可能性がある拡張子の圧縮・解凍についてコマンドを紹介します。  
一部のコマンドは環境によってはデフォルトで対応しておらず、別途コマンドをインストールした上で利用する必要があります。

## 1. .tar.gzもしくは.tgz
### 1-1. 解凍
```shell
tar -xzvf <解凍ファイル名>
```
### 1-2. 圧縮
```shell
tar -zcvf <ファイル名>.tar.gz <圧縮ファイル名>
```
## 2. .tar.bz2もしくは.tbz
### 2-1. 解凍
```shell
tar -xjvf <解凍ファイル名>
```
### 2-2. 圧縮
```shell
tar -jcvf <ファイル名>.tar.bz2 <圧縮ファイル名>
```
## 3. .gz
### 3-1. 解凍
```shell
gunzip <解凍ファイル名>
```
もしくは
```shell
gzip -d <解凍ファイル名>
```
### 3-2. 圧縮
```shell
gzip <圧縮ファイル名>
```
## 4. .bz2
### 4-1. 解凍
```shell
bzip2 -d <解凍ファイル名>
```
### 4-2. 圧縮
```shell
bzip2 <圧縮ファイル名>
```
## 5. .lhaもしくは.lzh
### 5-1. 解凍
```shell
lha x <解凍ファイル名>
```
### 5-2. 圧縮
```shell
lha c <ファイル名>.lzh <圧縮ファイル名>
```
## 6. .zip
### 6-1. 解凍
```shell
unzip <解凍ファイル名>
```
### 6-2. 圧縮
```shell
zip <ファイル名>.zip <圧縮ファイル名>
```
