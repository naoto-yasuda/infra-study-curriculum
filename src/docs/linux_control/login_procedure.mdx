# Linuxサーバーへのログイン手順
## 1.通常のログイン手順
### 1-1.TeraTermを起動する
インストールしたソフトウェアから「TeraTerm」を起動する
### 1-2.接続情報の入力
以下のように設定情報を入力して「OK」をクリックする

| 設定項目 | 値 |
| ------- | ------- |
| ホスト | xxx.xxx.xxx.xxx |
| サービス | SSH |
| TCPポート | 22 |
| SSHバージョン | SSH2 |
| IPバージョン | AUTO |
![](../login_procedure_linux_login_1.png)  
※接続情報の修正が必要

### 1-3.認証情報の入力
以下のように設定情報を入力して「OK」をクリックする

| 設定項目 | 値 |
| ------- | ------- |
| ユーザ名| xxxxxxxx |
| パスフレーズ | yyyyyyyy |
| 認証方式 | プレインパスワードを使う |
![](../login_procedure_linux_login_2.png)

### 1-4.ログイン確認
ログインできたことを確認する
![](../login_procedure_linux_login_3.png)


## 2.EC2サーバーの場合のログイン手順
### 2-1.TeraTermを起動する
インストールしたソフトウェアから「TeraTerm」を起動する
### 2-2.接続情報の入力
以下のように設定情報を入力して「OK」をクリックする

| 設定項目 | 値 |
| ------- | ------- |
| ホスト | xxx.xxx.xxx.xxx |
| サービス | SSH |
| TCPポート | 22 |
| SSHバージョン | SSH2 |
| IPバージョン | AUTO |
![](../login_procedure_linux_login_1.png)

### 2-3.認証情報の入力
以下のように設定情報を入力して「OK」をクリックする

| 設定項目 | 値 |
| ------- | ------- |
| ユーザ名| ec2-user |
| パスフレーズ | 空欄 |
| 認証方式 | RSA/DSA/ECDSA/ED25519鍵を使う |
| 秘密鍵 | ダウンロードしたpemキーファイルのパス |
![](../login_procedure_linux_ec2_login_2.png)

### 2-4.ログイン確認
ログインできたことを確認する
![](../login_procedure_linux_login_3.png)
