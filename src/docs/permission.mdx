# 権限の指定方法について
権限は
```
1. ファイルの所有者に対する権限
2. ファイルの所有グループに対する権限
3. その他に対する権限
```
の３種類があり、それぞれの権限に対して
```
1. 読み込み権限
2. 書き込み権限
3. 実行権限
```
の３種類の権限を付与することができます。

Linuxではこれらの権限を各種コマンドで変更することができ、その際権限を数字またはアルファベットで指定することができます。  
指定方法は以下のような形になります。  

| 数値 | アルファベット | 権限内容 |
| -------- | -------- | -------- |
| 0 | 文字なし | 権限なし |
| 1 | x | 実行権限のみ |
| 2 | w | 書き込み権限のみ |
| 3 | wx | 書き込み権限と実行権限 |
| 4 | r | 読み込み権限のみ |
| 5 | rx | 読み込み権限と実行権限 |
| 6 | rw | 読み込み権限と書き込み権限 |
| 7 | rwx | すべての権限 |

これらを３種類の権限ごとに組み合わせて各種コマンドでは利用します。  
```
例：ファイルの所有者にすべての権限を、他の対象は読み込み権限を付与する場合
・数字での指定
744
※解説：
　１文字目がファイルの所有者に対する権限
　２文字目がファイルの所有グループに対する権限
　３文字目がその他に対する権限

・アルファベットでの指定
u=rwx,g=r,o=r
※解説：
 u=でファイルの所有者に対する権限の指定
 g=でファイルの所有グループに対する権限の指定
 o=でその他に対する権限の指定
```
